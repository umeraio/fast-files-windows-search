What is Everything?

Everything is a search engine for Windows that locates files and folders by filename instantly. Unlike the built-in Windows search, it creates a real-time index of your drives, allowing you to find any file in milliseconds. You can search by filename, extension, or even file size.

Installation: Download the latest version from the official support page: voidtools.com/support/everything.

Important Note: The commands below use a "Deep Shield" to bypass Windows, Program Files, and AppData folders. This keeps your results clean and protects system files. If you wish to search these areas or add more folders, simply edit the !path: section.

Pro Exclusion Search

Find MP4s (Excluding Windows & System Clutter)

Perfect for finding your movies while ignoring thousands of OS-generated cache files.

ext:mp4 !path:windows !path:system
Image Cleanup

Duplicate Image Names (Safe)
